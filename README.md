# backoffice

## Dependências para Desenvolvimento

* Yarn >= 1.22.5
* IDE de sua preferência (Visual Studio Code, Sublime, etc)

---

## Instalação

    ```
    yarn install
    ```

## Rodar a aplicação

    ```
    yarn serve
    ```

## Comandos úteis

### Rodar testes

    ```
    yarn test:unit
    ```

### Rodar linter e corrigir bugs

    ```
    yarn lint
    ```
